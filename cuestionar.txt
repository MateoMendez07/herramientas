¿Para que sirve el git branch -a?
Se utiliza para listar todas las ramas en un repositorio Git, incluidas las ramas locales y remotas.

¿Cómo puedo ver la versión del git?
git -v

¿Cómo puedo clonar un proyecto en GitHub?
Con el comando git clone y la url

¿Para que sirve el comando git add .?
Mueve los cambios del directorio de trabajo al área del entorno de ensayo. Así puedes preparar una instantánea antes de confirmar en el historial oficial.
